¿Cúal fue el primer ratón del mundo?

El primer ratón del mundo, evidentemente en el ambito informatico, fue diseñado y construido en 1964 por Douglas Engelbart y estaba hecho de madera.

¿A qué hace referencia la sigla CAPTCHA?

La sigla CAPTCHA hace referencia a "Completely Automated Public Turing Test to tell Computers and Humans Apart".

¿Cúal fue el primer virus informatico?

El primer virus informatico se creo en 1971 para el sistema ARPANET, se llamaba "Creeper" y era inofensivo. Fue un experimento realizado para ver como este virus se extendia de un ordenador a otro.
